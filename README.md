This project focuses on analyzing sales performance and customer reviews using SQL and Power BI. The goal is to extract meaningful insights such as sales trends, top-performing states, customer satisfaction, and delivery performance.

⸻

🛠️ Tools & Technologies Use	
  •	Power BI
	•	Excel

⸻

📁 Dataset Description

The dataset includes:
	•	Order Date & Ship Date
	•	Sales & Quantity
	•	Customer ID
	•	State-wise data
	•	Review Points & Review Status

⸻

📊 Key Features of Dashboard
	•	Total Sales, Orders, and Quantity KPIs
	•	Current Year vs Last Year Sales Comparison
	•	Top Performing States by Sales
	•	Review Analysis (Good / Average / Bad)
	•	Monthly Sales Trends
	•	Average Processing & Delay Days
	•	Delivery Performance Insights

⸻

📈 Key Insights
	•	Identified top-performing states contributing the highest sales
	•	Analyzed customer satisfaction through review scores
	•	Found delivery delays and processing time trends
	•	Compared year-over-year sales growth
